Modelo de Caja en CSS y HTML

El modelo de caja en CSS y HTML es un concepto fundamental que define cómo se representan y se comportan los elementos en una página web. Cada elemento HTML se considera una caja rectangular, y el modelo de caja describe cómo se calcula el tamaño total de la caja y cómo se distribuyen los elementos dentro de ella.

Width y Height ➡️ ancho y alto.

La propiedad width determina el ancho del contenido de la caja, mientras que la propiedad height determina la altura del contenido. Estas propiedades no incluyen el padding, margin o border del elemento, solo el contenido interno.

Padding ➡️ Distancias internas / borde al contenido.

El padding es el espacio entre el contenido de la caja y su borde. Se puede definir usando la propiedad padding o las propiedades padding-top, padding-right, padding-bottom y padding-left por separado.

Margin ➡️ son las distancias externas / entre el HTML y sus elementos hermanos o padres.

El margin es el espacio entre una caja y otras cajas cercanas. Se puede definir usando la propiedad margin o las propiedades margin-top, margin-right, margin-bottom y margin-left por separado. Los márgenes se colapsan cuando dos elementos con márgenes se superponen, tomando el valor del margen más grande.

Border ➡️ Lo que delimita c/u de los tags

El border es el borde que rodea la caja. Se puede definir utilizando la propiedad border o sus propiedades específicas como:

  1. border-width - Ancho
  2. border-color - Color
  3. border-style - Estilos
El border separa el contenido de la caja del margen externo y se representa como una línea sólida, punteada, doble o de otros estilos.
  • none: No se muestra ningún borde.
  • hidden: Igual que none, pero se comporta de manera diferente en ciertos casos.
  • dotted: Crea un borde punteado.
  • dashed: Crea un borde con líneas discontinuas.
  • solid: Crea un borde sólido.
  • thin: Crea un borde de 1px.
  • medium: Crea un borde de 3px.
  • double: Crea un borde doble con dos líneas sólidas.
  • groove: Crea un borde tallado en 3D.
  • ridge: Crea un borde en relieve en 3D.
  • inset: Crea un borde con apariencia de estar hundido.
  • outset: Crea un borde con apariencia de estar sobresalido.
